Slider/Range

Slider are input components used to select a value inside a specific range.
You can also decide to select a range instead of a single value by switching from a single value Slider to a Range component.

Slider (Mono Value)

As explained above the slider component is used to select a single value out of a set range of numbers. The user can determine the selected value by sliding the sliders thumb between both extremes (minimum and maximum). The component also consists of two labels which, if wanted, are also able to be hidden.

Range (Range-Selection and Dual-Value)

Compared to Slider components Range components consist of two thumbs instead of one. Those two thumbs are able to both select two values separate each other, but also a range defined by the distance between both range thumbs - the range value is always absolute and therefore will only remain positive.

Code & Examples

To layout the range of the component simply add the component and set range to true.

Following interactions are available:

  • To hide both extremes (minimum and maximum) as labels add [showLabels] and set it to false.
  • To enable the range component add [showLabels] and set it to true.
  • To set the size of either Range or Slider add [size] and choose between small('sm') , medium('md') , large('lg') or xlarge('xl').
  • To react to value changes, bind to event (onValueChanged).
  • To react to low-Value changes (lower range-value), bind to event (onLowValueChanged).
  • To react to high-Value changes (higher range-value), bind to event (onHighValueChanged).

Slider components come in all different kind of sizes. The enum SliderSize is able to offer you four different kinds of sizes. The options are either small('sm') , medium('md') , large('lg') or xlarge('xl') . Also Range components come in the same sizes.
